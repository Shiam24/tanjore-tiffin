# Tanjore Tiffin — Demo Website

A demo South Indian restaurant website built with plain HTML, CSS and JavaScript.

## Features
- Filter coffee & tiffin themed design
- Animated kolam (rangoli) drawing in the hero section
- Tabbed menu (Tiffin / Meals / Kaapi)
- Photo gallery
- Customer review carousel
- Fully responsive

## Files
- `index.html` — page structure
- `style.css` — styling
- `script.js` — interactivity (kolam animation, tabs, carousel)

## How to view
Just open `index.html` in any browser.
